## Description

_In your own words, what is this paper about?_

This paper seeks to find out if a translation done with an AI system like GPT-4o has a distinct style that can be identifiable by using stylometry. This way, they could see if the translations done by AI can be identified and differentiated from translation done by professional translators. To do so, the researchers decided to test this by translating religious documents from Arabic to English. After getting the translated texts from both AI and the translators, the researchers found that the translations generated by GPT-4o demonstrate a distinctive style that can be used as indicators of AI generated translations. With the researchers noticing that this text had a type of formulaic approach, taking into account the frequency of the words and the number of stop words used.
