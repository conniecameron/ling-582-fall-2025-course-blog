## Project description

_See [the rubric](https://parsertongue.org/courses/snlp-2/assignments/rubrics/course-project/final/#reproducibility).  Use subsections to organize components of the description._

Draft summary:

For this project I am trying to do Optical Character Recognition on typed resources (old and typewritten, scanned documents) in Coeur d'Alene (CdA). The documents are written in a sort of phonetic transcription with lots of accents.
But the goal is we want to be able to extract the text, make it searchable, and able to be put into a database. First we have to extract it which I'm using transformers and synthetic data.

  Tools I'll be using:
  * Markovify (create made up words)
  * Pillow (generate the images)
  * Augraphy (for photo augmentation)

**Goal 3 (Nov 28)**: Improve the model and start working on layout parsing to seperate the English from the CdA and run the respective OCR models.
