1. Under the **Assignments** section of the course's D2L page, click the link in the `class-competition-code` assignment's description.

2. See the README in your repository for an invitation link to the competition.
  - The competition page provides an overview of the competition 

3. Use the provided `class-competition-code` assignment repository to manage your code for the class-wide competition (see the **Assignments** section of the course D2L page for the link)

4. Prepare a summary a blog-style summary of your approach and results ([see the README in this repo for details](https://github.com/uazhlt-ms-program/ling-582-fall-2025-course-blog#2-class-wide-competition-kaggle))

## Requirements

Minimally, your summary should include ...

- a description of your approach
- a summary of your results
- a link to the `class-competition-code` repository
- clear instructions for reproducing your results
- a proposal for future improvements
