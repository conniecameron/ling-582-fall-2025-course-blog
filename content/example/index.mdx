MDX supports extended [markdown](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax), HTML, and JSX.

## Adding images

You can use HTML `<img>` tags to add images to your posts:

```html
<img src="images/lou-reed.jpg" alt="Lou Reed (image from en.wikipedia.org/wiki/Lou_Reed)"/>
```

<img src="images/lou-reed.jpg" alt="Lou Reed (image from en.wikipedia.org/wiki/Lou_Reed)"/>


Alternatively, you can use markdown syntax:

```markdown
!["Lou Reed (image from en.wikipedia.org/wiki/Lou_Reed)"](images/lou-reed.jpg)
```


!["Lou Reed (image from en.wikipedia.org/wiki/Lou_Reed)"](images/lou-reed.jpg)

Note that with markdown syntax, you won't be able to easily resize the image.


## Adding math

You can include formulae using LaTeX math mode syntax:

```latex
$$\text{Softmax}(x_{i}) = \frac{\exp(x_{i})}{\sum_{j} \exp(x_{j})}$$
```

$$\text{Softmax}(x_{i}) = \frac{\exp(x_{i})}{\sum_{j} \exp(x_{j})}$$
